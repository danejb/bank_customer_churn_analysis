# Data Source

## Dataset Information

**Name:** Bank Customer Churn Dataset
**Source:** Kaggle
**Records:** 10,000 customers
**Fields:** 18 columns including Age, Tenure, Geography, Churn Status, Balance, etc.

---

## How to Access

1. Visit Kaggle: https://www.kaggle.com/
2. Search for "Bank Customer Churn"
3. Download the dataset

---

## Key Fields Used

| Field | Meaning |
| --- | --- |
| Age | Customer age (18-96 years) |
| Tenure | Years with bank (0-10 years) |
| Geography | Customer country (France, Germany, Spain) |
| Gender | Customer gender (Male/Female) |
| Exited | **Key:** Customer churn status (1=churned, 0=active) |
| Balance | Account balance |
| NumOfProducts | Number of products customer uses |
| CreditScore | Customer credit score |
| EstimatedSalary | Estimated annual income |
